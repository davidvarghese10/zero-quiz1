The website is Zero Points Quiz, an interactive, full-featured game controller and leaderboard designed for the unique "0 Points" Quiz Event.
Here is a breakdown of what the application does and how it is structured:
🎮 The Core Concept: Race to Zero
Unlike traditional trivia games where teams compete to gain the highest score, the objective of the Zero Points Quiz is the exact opposite:
The Goal: Teams start with a high point value (e.g., 200 points) and must work their way down to exactly 0 points.
Points Formula:
Correct Answers subtract points from a team's score.
Incorrect Answers or Passed Questions can add or maintain points depending on the rule parameters.
The team closest to exactly 0 points (without going below or remaining too high) ranks highest on the leaderboard.
✨ Key Features & User Experience
Neon Green-Blue Aesthetic: Styled with a high-contrast futuristic visual theme, incorporating smooth glassmorphism (glass panels, liquid buttons) and neon teal highlights.
Smart Dual-Mode Timers: Each question detail view features a dynamic, fully-interactive countdown timer that automatically adapts to the question's difficulty level:
30 Seconds for easy questions (40 points or less).
60 Seconds for hard questions (greater than 40 points).
Controllers to Pause, Resume, and Reset the clock on the fly.
Auto-Standings Projection: After any question is answered or passed, the app immediately redirects to the Standings tab to show live leaderboard updates, building suspense for the audience.
Presenter & Administrator Controls:
Main Board: A grid of questions sorted by categories and point weights.
Leaderboard View: Ranks active teams dynamically with indicators showing whose turn it currently is.
Coordinator Dashboard: A hidden administrative control panel allowing organizers to manually adjust points, override turns, reset the board, or edit questions.
